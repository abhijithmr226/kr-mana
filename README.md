# KR Thekkedathu Mana Resort Website

A premium, fully responsive multi-page website for **K.R. Thekkedathu Mana**, a traditional Ayurvedic heritage resort and event venue situated in Thrissur, Kerala.

## Features

- **Design System:** Sleek, high-end heritage-themed design system using warm cream and royal gold accents.
- **Dynamic Booking Calculator:** Automatically estimates stay costs on the client side based on check-in, check-out, and suite selection.
- **WhatsApp Booking Redirect:** Instantly formats guest booking parameters into a professional template and redirects to the resort's WhatsApp desk.
- **Interactive Lightbox Gallery:** Implements slide-show modal overlays for high-resolution downloaded resort photos.
- **SEO Optimized:** Fully equipped with Open Graph, Twitter cards, meta descriptions, canonical URLs, and semantic HTML5 hierarchy.
- **Local Dev Server:** Built-in lightweight native Node.js static file server.

## Getting Started

1. Clone this repository.
2. Run the development server locally:
   ```bash
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
